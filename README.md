# KFS Generator App

Static GitHub Pages app for stakeholder validation.

Open `index.html` in a browser, or host this folder through GitHub Pages. No local server is required for quick preview.

Features:
- Enter loan parameters.
- Uses fixed test borrower data for generation.
- Click Generate KFS to preview the generated KFS.
- View the selected language JSON after generation.
- Download/save the generated KFS as PDF through the browser PDF flow.
